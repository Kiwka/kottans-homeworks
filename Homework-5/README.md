#### Задача №1. 
Реализация функции deepCopy - для копирования объекта с учетом вложенных объектов:
var a = {b: ‘c’, d: {e: ‘f’}},
    b = deepCopy(a);
a.d = 12;
b.d // {e: ‘f’}
тесты: http://plnkr.co/edit/jhc1oSkWg7g87W2gjfiq?p=preview
